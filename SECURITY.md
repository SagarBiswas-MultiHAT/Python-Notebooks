# Security Policy

## Supported Content

This repository publishes educational study material and a static website. It does not ship executable security tooling.

## Responsible Use

The cybersecurity notebook discusses defensive concepts, automation patterns, and security tooling in an educational context. Use any ideas from the material only in environments where you have explicit authorization.

Do not use the material to target real systems, collect credentials, monitor devices without consent, or bypass access controls.

## Reporting a Security Concern

If you find a repository issue that could expose readers to unnecessary risk, open a GitHub issue describing:

- The affected file or page.
- The risk or misleading guidance.
- A safe recommendation for fixing it.

If the concern should not be discussed publicly, contact the repository owner privately before opening an issue.
